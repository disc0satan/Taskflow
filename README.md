## üöÄ Features
* **Custom Categories:** Add your own tags with a built-in color picker to create unique glows.
* **Filtering System:** Toggle between `Work`, `Personal`, `Social`, or your custom-defined tags.
* **Persistent Storage:** Uses `localStorage` so your data survives a page refresh.
* **Task Logic:** Separate "Done" (crossed out/dimmed) and "Delete" actions to track your progress.
* **Zero-Task Reward:** Special "System Override" message when all tasks are completed.
